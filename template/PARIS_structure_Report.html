Analysis Overview

This report summarises the results of PARIS (Psoralen Analysis of RNA Interactions and Structures) sequencing data processed through the EasyOmics PARIS pipeline in structure mode. The analysis identifies RNA duplex groups (DGs) from chimeric reads to reveal intramolecular and intermolecular RNA secondary structures.

A total of {{ num_samples }} sample(s) were processed. Reads were adapter-trimmed, deduplicated, and mapped to the reference genome using STAR with chimeric read detection enabled. Duplex groups were assembled using samPairingCalling.

Duplex Group Analysis

About Duplex Groups

Duplex groups (DGs) represent sets of chimeric reads that support a specific base-pairing interaction within an RNA molecule. Each DG captures an RNA structural element from the PARIS experiment. Higher-confidence structures are supported by more read pairs.

Methods

Analysis Pipeline

PARIS data were processed using the EasyOmics PARIS Processing Pipeline (structure mode) implemented in Snakemake. The pipeline includes:

  1. Adapter Trimming – 3′ adapter sequences removed with Trimmomatic; minimum length {{ min_len_3 | default(28) }} nt.
  2. Read Deduplication – PCR duplicates collapsed with readCollapse.
  3. 5′ Trimming – HEADCROP (17 nt) and length filtering (min {{ min_len_5 | default(20) }} nt) with Trimmomatic.
  4. Quality Control – Raw and trimmed reads assessed with FastQC; results aggregated with MultiQC.
  5. Genome Alignment – Reads mapped to the reference genome with STAR, chimeric read detection enabled (chimSegmentMin={{ chim_segment_min | default(15) }}).
  6. Duplex Group Assembly – Chimeric alignments assembled into duplex groups using samPairingCalling (minLen={{ dg_minlen | default(15) }}, minPair={{ dg_minpair | default(2) }}).
  7. NG Assembly – Near-gapless alignments extracted with sam2ngmin.py.
  8. BED Conversion – DGs converted to BED12 format with dg2bed.py.
  9. Alternative Structure Analysis – Competing conformations identified with alternativestructure.py.
